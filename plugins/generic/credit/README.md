# CRediT plugin for OJS

This plugin adds support for the NISO CRediT contributor role standard in OJS 3.3/3.4/3.5 and OPS 3.5.
It was started at the 2022 Helsinki sprint.

As of OJS 3.6 and later, CRediT contributor roles have been integrated into the core.

Working group:

- Esko Clarke Sario: Finnish Lifelong Learning Foundation (Kvs)
- Alec Smecher: Public Knowledge Project
